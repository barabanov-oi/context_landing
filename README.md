# context_landing

SaaS-приложение на Flask: лента кейсов PPC-специалиста + личный кабинет с быстрым email/password входом и подключением аккаунтов Яндекс Директ через API.

## Что есть

- Лента кейсов на `/`.
- Отдельная страница каждого кейса на `/cases/<slug>`.
- Личный кабинет:
  - быстрая регистрация на `/signup`;
  - вход на `/login`;
  - подключение аккаунтов Яндекс Директ через API (`customers.get`);
  - просмотр и отключение подключённых аккаунтов.
- Админка кейсов на `/admin` (защищена паролем):
  - добавление нового кейса;
  - редактирование существующего кейса;
  - визуальный HTML-редактор для кастомного контента кейса.
- Данные хранятся в:
  - `data/cases.json` — кейсы;
  - `data/users.json` — пользователи и их подключения к Директу.

## Настройки

- `ADMIN_PASSWORD` — пароль админки (по умолчанию `admin123`).
- `FLASK_SECRET_KEY` — секрет Flask-сессии.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install flask requests
export ADMIN_PASSWORD='ваш_пароль'
flask --app app run
```

После запуска откройте `http://127.0.0.1:5000/`.
